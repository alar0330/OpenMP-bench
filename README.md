# OpenMP-bench

OpenMP Benchmarks of matrix multiplication for Fortran.

TODO: also consider BLAS and OpenBLAS

Setup:
```
>> setx NUM_THREADS(4)
>> gfortran -Wall -fopenmp -O0 -o test_gemm.exe multiplex.f90 test_gemm.f90
```
